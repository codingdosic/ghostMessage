## 2. 상세 설명 (Detailed Description)

### (KR) 소개
**"웹 전체를 하나의 거대한 소셜 레이어로 만듭니다."**

Web Ghost Messages는 다크소울이나 엘든링의 메시지 시스템에서 영감을 받은 브라우저 확장 프로그램입니다. 웹 서핑 중 특정 링크 위에 다른 탐험가들이 남긴 "유령 흔적(메시지)"을 발견할 수 있습니다. 

누군가 먼저 지나간 길의 힌트를 얻거나, 텅 빈 웹사이트에 작은 발자취를 남겨보세요.

### ✨ 주요 기능
- **메시지 남기기**: 아무 링크 위에서 우클릭하여 나만의 메시지를 남길 수 있습니다.
- **실시간 발견**: 다른 사용자가 남긴 메시지가 있는 링크는 은은한 하이라이트로 표시됩니다.
- **툴팁 미리보기**: 하이라이트된 링크 위에 마우스를 올리면 메시지를 즉시 확인할 수 있습니다.
- **투표 시스템**: 도움이 되거나 재미있는 메시지에는 추천(👍)과 비추천(👎)을 눌러 상호작용하세요.
- **다크모드 지원**: 눈이 편안한 다크모드 설정 페이지를 제공합니다.

### 🚀 시작하기
1. 익스텐션을 설치합니다.
2. 아무 웹사이트나 방문하여 링크(<a> 태그)를 찾아보세요.
3. 링크 위에서 **우클릭 > Leave a Ghost Message...**를 클릭하여 메시지를 작성합니다.
4. 하이라이트된 링크를 발견하면 마우스를 올려보세요!

### (EN) Introduction
**"Turn the entire web into a giant social layer."**

Web Ghost Messages is a browser extension inspired by the asynchronous multiplayer message systems of games like Dark Souls and Elden Ring. Discover "ghost traces" (messages) left by other explorers on specific links as you surf the web.

Get hints for a path someone has already taken, or leave your own small footprint on a lonely website.

### ✨ Key Features
- **Leave Messages**: Right-click on any link to leave your own message.
- **Real-time Discovery**: Links with messages left by others are marked with a subtle highlight.
- **Tooltip Preview**: Hover over a highlighted link to instantly read the messages.
- **Voting System**: Interact with helpful or funny messages by giving them a Thumbs Up (👍) or Down (👎).
- **Dark Mode Support**: A stylish dark mode is available for the settings page.

## 3. 콘텐츠 관리 정책 (Content Policy)

**건전한 커뮤니티 유지를 위해 아래에 해당하는 메시지는 사전 고지 없이 삭제될 수 있습니다.**
- 타인에 대한 비방, 욕설, 혐오 표현이 포함된 경우
- 무분별한 광고, 스팸, 불법 정보 홍보
- 타인의 개인정보를 노출하는 행위
- 기타 서비스 운영 취지에 맞지 않는 부적절한 콘텐츠

**Messages that fall under the following categories may be deleted without prior notice to maintain a healthy community.**
- Hate speech, harassment, or profanity
- Excessive advertising, spam, or promotion of illegal information
- Exposure of others' personal information
- Any other content deemed inappropriate for the service

## 4. 개인정보 처리방침 (Privacy Policy)

**Web Ghost Messages**는 사용자의 프라이버시를 존중합니다.
- **수집 데이터**: 
    1. 익명 사용자 식별자 (UUID)
    2. 사용자가 직접 작성한 메시지 내용 및 대상 URL
- **사용 목적**: 메시지 표시 및 투표 기능 제공을 위해서만 사용됩니다.
- **공유 금지**: 수집된 정보는 제3자에게 판매하거나 공유하지 않습니다.

## 4. 권한 사용 사유 (Justification for Permissions)

*   **activeTab**: 현재 탭의 링크들을 스캔하여 유령 메시지 하이라이트를 표시하기 위해 사용됩니다.
*   **contextMenus**: 링크 위에서 우클릭 시 메시지 작성 메뉴를 노출하기 위해 필요합니다.
*   **storage**: 사용자 설정(테마, 정렬 기준 등) 및 익명 사용자 식별자를 로컬에 저장하기 위해 사용됩니다.
