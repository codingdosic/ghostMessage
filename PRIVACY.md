# 개인정보처리방침 (Privacy Policy)

**Web Ghost Messages**는 사용자의 개인정보를 중요하게 생각하며, 서비스를 제공하는 데 필요한 최소한의 데이터만을 수집합니다.

## 1. 수집하는 데이터
당사는 서비스를 제공하기 위해 다음과 같은 데이터를 수집합니다.
- **익명 사용자 식별자 (UUID)**: 사용자 기기를 식별하여 메시지 작성 및 투표 기능을 제공하기 위해 사용됩니다.
- **메시지 내용**: 사용자가 웹페이지에 직접 작성한 텍스트 데이터입니다.
- **URL 정보**: 사용자가 메시지를 남기거나 조회하는 대상 웹페이지의 주소입니다.

## 2. 데이터 사용 목적
- 웹페이지 내 메시지 표시 및 공유
- 투표(추천/비추천) 기능 구현
- 사용자 설정(다크모드 등) 유지

## 3. 데이터 공유 및 판매
당사는 수집된 정보를 제3자에게 판매하거나 공유하지 않습니다. 데이터는 오직 서비스 기능 구현을 위해서만 사용됩니다.

## 4. 데이터 삭제
사용자는 언제든지 확장 프로그램 설정 또는 기기 내 저장소 삭제를 통해 로컬 데이터를 관리할 수 있습니다. 서버 데이터 삭제와 관련된 문의는 아래 이메일로 요청해 주세요.
